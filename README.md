# autocomplete

Web service which provides the autocomplete feature from the given word.

## Installation
**Copy a project**
```
$ git clone git@github.com:Dddarknight/autocomplete.git
$ cd autocomplete 
```

**Launch**
```
$ go run cmd/web/*.go
```
